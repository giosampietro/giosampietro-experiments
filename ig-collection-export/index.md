# IG Collection Export

IG Collection Export is an experimental Chrome extension for saving your own Instagram Saved collections to a local folder you choose.

Install link coming after Chrome Web Store review.

## What It Does

- Saves images to a local folder selected by you
- Runs only when you start an export
- Has no analytics or telemetry
- Does not send exported files to the author or an author-controlled server

## Current Limitations

- Video export is not supported
- Exports are intentionally paced and may take time
- Individual support is not offered
