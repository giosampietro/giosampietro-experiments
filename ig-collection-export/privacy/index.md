# Privacy Policy

Last updated: 2026-05-06

Legal version: 1

IG Collection Export is designed as a local personal archive tool. The extension has no analytics or telemetry.

The extension does not send data to the author or any author-controlled server. It accesses Instagram and Instagram-associated media hosts only as needed for the export you start.

## Data Handled Locally

When you start an export, the extension may handle these categories of data locally in Chrome or in the folder you select:

- Instagram collection URL, name, and collection identifier
- Post links and shortcodes
- Usernames shown in filenames or export logs
- Post dates used in filenames
- Image media URLs needed to save files locally
- Export progress and visible export log state
- Selected folder handle and folder confirmation state
- Local recovery data in an `export_memory` file
- Saved image files

These data categories are used to perform the export, name files, avoid duplicate work, and recover progress after interruption.

## Local Files

The extension can write only to the folder you select for export, subject to browser-granted access.

The extension saves image files and local recovery data to the folder you select. It is not designed to delete your existing files.

The extension does not overwrite existing non-empty image files.

## Retention

The author does not retain extension data because it is not sent to the author.

You can delete exported files from the folder you selected and clear extension data through Chrome.

For privacy questions, contact experiments@giosampietro.xyz.
